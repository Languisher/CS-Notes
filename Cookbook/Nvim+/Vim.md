# Vim

> I highly recommend the book "Practical Vim: Edit Text at the Speed of Thought".

## Essentials of Vim

- Use `.`+`{deplacement}`(such as `j`) to repeat your actions. No matter what repeating is a dull and meaningless work.
- Regularly drop out from INSERT mode.

- `{Operator}+{Motion}`, such as `guiw`, transforming a (inner) word to all lower-capped alphabets.

- In order to allow `.` operation to repeat your actions, use the operator actions when it is possible, rather than entering the visual mode. Use visual mode when it is hard to realize using normal operator motions, for example entering text in the same position of various lines.

- **Command line mode** is suitable to do opertaions within a large scale or apart from a huge distance, such as `:14t$`. 
    - range: `{start},{end}`
        - `.` current line
        - `%` all lines
        - `/<pattern>/` find with patterns, example: `:/,/,/possible`
    - `:[range]t{address}`(`:6,7t.`) copy a line to the position of address
    - `:[range]m{address}`
    - Repeat a command(like `.`) to multiple lines: `:[range]normal {command}`
    - `Tab` to autofill
    - `!` to call shell commands
    - `@:` to **repeat command line commands**

- **Different files**: 注意缓冲区（小）和标签页（大）的区别
    - `<C-^>` 当前文件和轮换文件切换，`:bnext` + `@:` 在所有缓冲区间切换
    - 删除缓冲区文件（即上方条）: `:[num] bd` (use `:ls` first to find the `num`)
    - `<C-w>c`, `<C-w>v`, `<C-w>c`, `sp`, `vsp`
    - 创建/切换标签页：`:tabe[dit] {filename}`, `gt` or `:tabn[ext]`

- **Open and save file**: 
  
- Visual Mode -> command line mode, `:normal`

## Markdown in Vim

- `zr`: reduces fold level throughout the buffer
- `zR`: opens all folds
- `zm`: increases fold level throughout the buffer
- `zM`: folds everything all the way
- `za`: open a fold your cursor is on
- `zA`: open a fold your cursor is on recursively
- `zc`: close a fold your cursor is on
- `zC`: close a fold your cursor is on recursively


## Solutions to a number of problems

- Problem with the indentation when pasting:
    - Solution: type `:set paste` then paste the text. `:set nopaste` to quit the `paste` environment.
    - Reference: [Vim中复制粘贴缩进错乱问题的解决方案](https://blog.csdn.net/wzy_1988/article/details/50264285)

vnoremap <silent> <F5> :<C-u>call SaveAndExecutePython()<CR>

function! SaveAndExecutePython()
    " SOURCE [reusable window]: https://github.com/fatih/vim-go/blob/master/autoload/go/ui.vim

    " save and reload current file
    silent execute "update | edit"

    " get file path of current file
    let s:current_buffer_file_path = expand("%")

    let s:output_buffer_name = "Python"
    let s:output_buffer_filetype = "output"

    " reuse existing buffer window if it exists otherwise create a new one
    if !exists("s:buf_nr") || !bufexists(s:buf_nr)
        silent execute 'botright new ' . s:output_buffer_name
        let s:buf_nr = bufnr('%')
    elseif bufwinnr(s:buf_nr) == -1
        silent execute 'botright new'
        silent execute s:buf_nr . 'buffer'
    elseif bufwinnr(s:buf_nr) != bufwinnr('%')
        silent execute bufwinnr(s:buf_nr) . 'wincmd w'
    endif

    silent execute "setlocal filetype=" . s:output_buffer_filetype
    setlocal bufhidden=delete
    setlocal buftype=nofile
    setlocal noswapfile
    setlocal nobuflisted
    setlocal winfixheight
    setlocal cursorline " make it easy to distinguish
    setlocal nonumber
    setlocal norelativenumber
    setlocal showbreak=""

    " clear the buffer
    setlocal noreadonly
    setlocal modifiable
    %delete _

    " add the console output
    silent execute ".!python3 " . shellescape(s:current_buffer_file_path, 1)

    " resize window to content length
    " Note: This is annoying because if you print a lot of lines then your code buffer is forced to a height of one line every time you run this function.
    "       However without this line the buffer starts off as a default size and if you resize the buffer then it keeps that custom size after repeated runs of this function.
    "       But if you close the output buffer then it returns to using the default size when its recreated
    "execute 'resize' . line('$')

    " make the buffer non modifiable
    setlocal readonly
    setlocal nomodifiable
endfunction

" 支持在Visual模式下，通过C-y复制到系统剪切板
vnoremap <C-y> "+y
" 支持在normal模式下，通过C-p粘贴系统剪切板
nnoremap <C-p> "*p
